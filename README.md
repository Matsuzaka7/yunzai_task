# YunzaiV3 - task

**仅适配 YunzaiV3**

****

## 关于本插件
- 实现类似闹钟的功能，添加任务后将会在指定时间后提醒
- 任务指令：
  - 添加任务：`[xx秒|xx分钟|xx小时|xx天]后提醒我 睡觉`
  - 查看任务：`定时任务列表`
  - 删除任务：`删除任务 + id`
- by 松坂砂糖

## 安装

#### 方式一：直接下载压缩包

下载js后复制到 `Yunzai-Bot/plugins/example` 目录下，无需重启可直接使用
